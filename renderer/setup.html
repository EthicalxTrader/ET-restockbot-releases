Auto-ordering
RestockBot can place orders automatically the moment it detects a restock. You control the safety level.
Enable auto-ordering
Automatically add items to cart and place orders when restocked.
⚠ Auto-ordering uses real money. Start with Dry Run mode and only disable it once you've verified everything works correctly.
Dry run mode (recommended)
Simulates orders and logs what would happen — no real money spent. Turn this off when you're ready to go live.
Require confirmation before ordering
Sends you a notification before placing any order. You get a window to cancel.
